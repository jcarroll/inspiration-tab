# Inspiration Tab

Replaces the chrome new tab page with beautiful images and inspirational quotes from General Conference and Book of Mormon scriptures.

## Description

Inspired by Mo'Tab (RIP) 

## Getting Started

1. Download files into a folder
1. chrome://extensions
1. Enable developer mode
1. Load unpacked extension
